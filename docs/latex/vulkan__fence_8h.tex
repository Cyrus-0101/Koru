\doxysubsection{Detailed Description}
Fence management utilities for the Koru Engine\textquotesingle{}s Vulkan renderer. 

This module provides functions to\+:
\begin{DoxyItemize}
\item Create and destroy Vk\+Fence objects
\item Wait for fences (CPU waits for GPU)
\item Reset fences for reuse
\end{DoxyItemize}

Fences are used during rendering to ensure safe access to resources like\+:
\begin{DoxyItemize}
\item Command buffers
\item Swapchain images
\item Memory allocations 
\end{DoxyItemize}
