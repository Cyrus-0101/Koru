\doxysubsection{Detailed Description}
Math utilities for the Koru Engine. 

This module wraps basic math functions from \texorpdfstring{$<$}{<}math.\+h\texorpdfstring{$>$}{>} to avoid importing it everywhere. It also provides random number generation functionality with optional range support.

All functions are safe to call before full engine initialization, except those that require seeded randomness.
