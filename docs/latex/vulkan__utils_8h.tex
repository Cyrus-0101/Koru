Utility functions for working with Vk\+Result values. 

This module provides\+:
\begin{DoxyItemize}

\item Helper function to check if a result is considered a success or failure

\begin{DoxyItemize}

\item Handle validation layer warnings
\item Debug GPU operations like image creation, command submission, etc.
