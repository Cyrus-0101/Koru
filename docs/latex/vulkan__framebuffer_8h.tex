\doxysubsection{Detailed Description}
Framebuffer management interface for the Koru Engine\textquotesingle{}s Vulkan renderer. 

This module provides functions to\+:
\begin{DoxyItemize}
\item Create framebuffers that bind image views to a render pass
\item Destroy them safely during shutdown
\end{DoxyItemize}

Framebuffers are tightly coupled to the swapchain and must be recreated when\+:
\begin{DoxyItemize}
\item The window resizes
\item The swapchain is re-\/created (e.\+g., due to vsync toggle or resolution change) 
\end{DoxyItemize}
