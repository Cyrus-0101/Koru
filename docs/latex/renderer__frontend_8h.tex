\doxysubsection{Detailed Description}
Frontend interface for the rendering system. 

This module provides the main entry points for initializing, shutting down, and driving the rendering system. It abstracts away the specific backend (e.\+g., Vulkan, Open\+GL) and offers a simple API to be used by the core engine.

The frontend works with render packets and handles frame updates, resizing, and drawing operations.
