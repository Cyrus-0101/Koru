\doxysubsection{Detailed Description}
Utility functions for Vulkan shader module creation. 

This module provides functions to load SPIR-\/V shader binaries from files, create Vulkan shader modules, and set up shader stage info for pipeline creation. It supports multiple shader stages such as vertex and fragment shaders.
