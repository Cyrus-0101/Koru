\doxysubsection{Detailed Description}
Vulkan device management interface. 

This module provides functions to\+:
\begin{DoxyItemize}
\item Query available physical devices
\item Select and create a logical device
\item Query swapchain support on a given device and surface
\end{DoxyItemize}

These functions abstract away platform-\/specific device selection logic, making it easier to integrate with higher-\/level rendering systems.
