\doxysubsection{Detailed Description}
A 3-\/component vector union with multiple access patterns. 

Provides access to components via\+:
\begin{DoxyItemize}
\item Position (x,y,z)
\item Color (r,g,b)
\item Texture coordinates (s,t,p)
\item Raw array index
\end{DoxyItemize}

Why needed\+:
\begin{DoxyItemize}
\item Standard representation for 3D positions and directions
\item Color operations without separate color structs
\item 3D texture coordinates support 
\end{DoxyItemize}
