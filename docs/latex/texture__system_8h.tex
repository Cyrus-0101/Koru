\doxysubsection{Detailed Description}
Texture management system. 

This module provides functionality to manage texture resources within the engine. It supports loading, acquiring, and releasing textures, as well as automatic cleanup of unused resources.

Usage\+:
\begin{DoxyItemize}
\item Initialize the texture system with {\ttfamily \doxylink{texture__system_8h_a5311316a2adfccd1ecadf29aea704cf0}{texture\+\_\+system\+\_\+initialize()}}, providing configuration options.
\item Acquire textures by name using {\ttfamily \doxylink{texture__system_8h_a94229d7d8c2e06fabb2d9e3c2296599e}{texture\+\_\+system\+\_\+acquire()}}. If the texture is not already loaded, it will be loaded from disk.
\item Release textures using {\ttfamily \doxylink{texture__system_8h_ab454daecfb4133cbca5559dd84db3ac4}{texture\+\_\+system\+\_\+release()}}. If a texture was marked for auto-\/release, it will be freed when no longer in use.
\item Shutdown the texture system with {\ttfamily \doxylink{texture__system_8h_a3170cfd4c3f2bbf15a69e6a965a39a03}{texture\+\_\+system\+\_\+shutdown()}} to free all resources. 
\end{DoxyItemize}
