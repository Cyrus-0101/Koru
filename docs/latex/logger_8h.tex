\doxysubsection{Detailed Description}
Public interface for the Koru Engine logging system. 

This module provides a flexible logging system with support for\+:
\begin{DoxyItemize}
\item Multiple severity levels (FATAL, ERROR, WARN, INFO, DEBUG, TRACE)
\item Compile-\/time log level filtering
\item Color-\/coded console output
\item Assertion failure reporting
\end{DoxyItemize}

Usage\+: Use the provided macros ({\ttfamily KFATAL}, {\ttfamily KERROR}, {\ttfamily KWARN}, etc.) to log messages at different severity levels.
