\doxysubsection{Detailed Description}
Implementation of the input system for keyboard and mouse events. 

This module implements a state-\/based input system that tracks\+:
\begin{DoxyItemize}
\item Current and previous keyboard states
\item Current and previous mouse states
\item Button presses, movement, and wheel scroll
\end{DoxyItemize}

Input updates are handled once per frame in {\ttfamily \doxylink{input_8c_a85450297ce513c03fe70c80cdba41609}{input\+\_\+update()}}, which copies current state to previous for delta detection.

Usage\+: Use the public interface defined in {\ttfamily \doxylink{input_8h}{input.\+h}} to interact with this system.
