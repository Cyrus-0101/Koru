\doxysubsection{Detailed Description}
Memory allocation system with tagging and tracking. 

This module introduces a tagged memory allocator that helps track\+:
\begin{DoxyItemize}
\item Where memory is being used (e.\+g., textures, entities, renderer, etc.)
\item Total allocations per tag
\item Potential leaks or misuses
\end{DoxyItemize}

Usage\+: Replace raw {\ttfamily malloc()}/{\ttfamily free()} calls with {\ttfamily \doxylink{kmemory_8h_a35f22575c2b4fcbda423b6108e352b15}{kallocate()}}/{\ttfamily \doxylink{kmemory_8h_a6b55b65a2f2d473dac0d8ff4605a44fb}{kfree()}}.
