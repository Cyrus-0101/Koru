\doxysubsection{Detailed Description}
Frontend implementation for the Koru rendering system. 

This file provides the implementation for the main rendering interface functions, abstracting away the specific backend (e.\+g., Vulkan, Open\+GL) from the rest of the engine. It manages initialization, frame lifecycle, and drawing operations using a render packet.
