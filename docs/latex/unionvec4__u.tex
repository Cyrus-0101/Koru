\doxysubsection{Detailed Description}
A 4-\/component vector union with multiple access patterns. 

Provides access to components via\+:
\begin{DoxyItemize}
\item Position (x,y,z,w)
\item Color (r,g,b,a)
\item Texture coordinates (s,t,p,q)
\item Raw array index
\item SIMD operations when enabled 
\end{DoxyItemize}
