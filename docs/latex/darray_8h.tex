\doxysubsection{Detailed Description}
Dynamic array implementation with automatic resizing. 

This module provides a flexible dynamic array (darray) system that can store any data type. It supports\+:
\begin{DoxyItemize}
\item Dynamic resizing based on capacity and growth factor
\item Insertion and removal at arbitrary indices
\item Compile-\/time type safety through macros
\item Tracking of length, capacity, and stride
\end{DoxyItemize}

Usage\+: Use the macro helpers like {\ttfamily \doxylink{darray_8h_ab9d093e09b642e0d480a7ae82bad3af5}{darray\+\_\+create(type)}} and {\ttfamily \doxylink{darray_8h_a7ad89931e6589fa09b83ea99de2b32e6}{darray\+\_\+push()}} to work with typed arrays. Never call internal {\ttfamily \+\_\+darray\+\_\+\texorpdfstring{$\ast$}{*}} functions directly.
