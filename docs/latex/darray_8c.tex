\doxysubsection{Detailed Description}
Implementation of a dynamic array container. 

This module implements a generic dynamic array (darray) system that supports\+:
\begin{DoxyItemize}
\item Dynamic resizing based on usage
\item Insertion and removal at arbitrary indices
\item Type-\/safe macro wrappers for use in C code
\item Memory tracking via MEMORY\+\_\+\+TAG\+\_\+\+DARRAY
\end{DoxyItemize}

The internal layout consists of\+:
\begin{DoxyItemize}
\item A header block containing metadata\+: capacity, length, stride
\item A contiguous block of elements stored after the header
\end{DoxyItemize}

Usage\+: This file should not be used directly. Instead, use the macro helpers defined in {\ttfamily \doxylink{darray_8h}{darray.\+h}}.
