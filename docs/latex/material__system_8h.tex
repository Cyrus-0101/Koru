\doxysubsection{Detailed Description}
Material management system. 

This module provides functionality to manage material resources within the engine. It supports loading, acquiring, and releasing materials, as well as automatic cleanup of unused resources.

Usage\+:
\begin{DoxyItemize}
\item Initialize the material system with {\ttfamily \doxylink{material__system_8h_acd96d4f865338dcf64ca455731b43718}{material\+\_\+system\+\_\+initialize()}}, providing configuration options.
\item Acquire materials by name using {\ttfamily \doxylink{material__system_8h_ae395c1f929cb52e4876eafa9b2398f66}{material\+\_\+system\+\_\+acquire()}}. If the material is not already loaded, it will be created with default properties.
\item Release materials using {\ttfamily \doxylink{material__system_8h_a71642cf5861f603f7e1bf6c4d373b906}{material\+\_\+system\+\_\+release()}}. If a material was marked for auto-\/release, it will be freed when no longer in use.
\item Shutdown the material system with {\ttfamily \doxylink{material__system_8h_ac54691336a811ff2d5c8dc06727fba89}{material\+\_\+system\+\_\+shutdown()}} to free all resources. 
\end{DoxyItemize}
