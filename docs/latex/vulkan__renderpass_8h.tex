\doxysubsection{Detailed Description}
Vulkan render pass management interface for the Koru Engine. 

This module provides functions to\+:
\begin{DoxyItemize}
\item Create and destroy render passes
\item Begin and end rendering operations within a render pass
\end{DoxyItemize}

Render passes define how rendering commands interact with attachments like\+:
\begin{DoxyItemize}
\item Color buffers (swapchain images)
\item Depth/stencil buffers
\item Subpasses and load/store operations 
\end{DoxyItemize}
