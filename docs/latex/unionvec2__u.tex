\doxysubsection{Detailed Description}
A 2-\/component vector union with multiple access patterns. 

Provides access to components via\+:
\begin{DoxyItemize}
\item Position (x,y)
\item Color (r,g)
\item Texture coordinates (s,t) or (u,v)
\item Raw array index
\end{DoxyItemize}

Why needed\+:
\begin{DoxyItemize}
\item Unified type for different interpretations of 2D data
\item Memory-\/efficient (no conversion between color/position/\+UVs needed)
\item Cleaner code with semantic naming (e.\+g., .x/.y for position vs .u/.v for UVs) 
\end{DoxyItemize}
