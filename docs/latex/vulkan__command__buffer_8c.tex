\doxysubsection{Detailed Description}
Implementation of Vulkan command buffer management utilities. 

This module implements all functions related to\+:
\begin{DoxyItemize}
\item Allocation from a command pool
\item Beginning and ending command buffer recording
\item Submission to queues
\item State tracking
\end{DoxyItemize}

These are central to how the engine records and executes GPU commands.
