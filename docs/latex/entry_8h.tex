\doxysubsection{Detailed Description}
Main entry point interface for the engine. 

This file defines the externally-\/defined {\ttfamily \doxylink{entry_8h_ac1205073da2ce08a82ead459fc8b39cb}{create\+\_\+game()}} function that must be implemented by client applications. It also contains the {\ttfamily \doxylink{entry_8h_a840291bc02cba5474a4cb46a9b9566fe}{main()}} function declaration which initializes and runs the game loop.

Responsibilities\+:
\begin{DoxyItemize}
\item Define the entry point contract
\item Ensure proper initialization order
\item Integrate memory and logging systems at startup 
\end{DoxyItemize}
