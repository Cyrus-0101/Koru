\doxysubsection{Detailed Description}
Configuration settings for initializing the application. 

This structure contains window-\/related configuration such as position, size, and application name. It is passed to \doxylink{application_8c_af0ccecbeef900f30b0b734eb020574ef}{application\+\_\+create()} during startup.
