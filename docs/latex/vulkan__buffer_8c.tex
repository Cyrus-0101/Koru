\doxysubsection{Detailed Description}
Implements Vulkan buffer management functions for the Koru Engine. 

This module provides functions to create, destroy, resize, bind, lock, unlock, and copy data to/from Vulkan buffers. It handles memory allocation and binding for buffers used in rendering operations.
