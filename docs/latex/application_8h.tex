\doxysubsection{Detailed Description}
Public interface for the core application system. 

This module provides the main entry point and lifecycle management for the engine. It handles\+:
\begin{DoxyItemize}
\item Application configuration (window size, position, name)
\item Initialization of core systems (memory, logging, platform layer)
\item The main game loop
\end{DoxyItemize}

Usage\+:
\begin{DoxyItemize}
\item Define a {\ttfamily \doxylink{entry_8h_ac1205073da2ce08a82ead459fc8b39cb}{create\+\_\+game()}} function that populates a {\ttfamily game} struct
\item Call {\ttfamily \doxylink{application_8h_aa05477b1f7e65b39439a7bacf89418d8}{application\+\_\+create()}} to initialize the engine
\item Call {\ttfamily \doxylink{application_8h_a08c8575858e12aa5260a1cc0101df8bd}{application\+\_\+run()}} to start the game loop 
\end{DoxyItemize}
