\doxysubsection{Detailed Description}
Implementation of Vulkan image management utilities. 

This module provides functions to\+:
\begin{DoxyItemize}
\item Create and allocate Vk\+Image objects
\item Generate Vk\+Image\+Views for use in render passes or shaders
\item Destroy images and release associated resources
\end{DoxyItemize}

These utilities are used throughout the engine for\+:
\begin{DoxyItemize}
\item Swapchain image setup
\item Depth/stencil buffers
\item Texture loading and management 
\end{DoxyItemize}
