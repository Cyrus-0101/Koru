\doxysubsection{Detailed Description}
Event system for inter-\/component communication. 

This module provides a generic event system that allows components to\+:
\begin{DoxyItemize}
\item Register listeners for specific event codes
\item Fire events with optional context data
\item Unregister listeners when no longer needed
\end{DoxyItemize}

Events are identified by a 16-\/bit code, and can carry up to 128 bytes of arbitrary data via the {\ttfamily \doxylink{structevent__context}{event\+\_\+context}} union. System-\/defined event codes use values up to {\ttfamily 0x\+FF}; applications should define their own codes starting from {\ttfamily 0x100} or higher.
