Detailed Description

Vulkan pipeline creation functions.

This file contains functions for creating graphics pipelines in Vulkan. It includes the main function to create a graphics pipeline with specified parameters.

Creates a Vulkan graphics pipeline with the specified parameters.

This function sets up a graphics pipeline including shader stages, vertex input, viewport, rasterization, multisampling, depth/stencil state, color blending, and dynamic states. It also creates the pipeline layout with provided descriptor set layouts and push constant ranges.

Parameters
contextA pointer to the active Vulkan context.
renderpassA pointer to the vulkan_renderpass defining the render pass to use.
attribute_countThe number of vertex input attributes.
attributesAn array of VkVertexInputAttributeDescription defining vertex attributes.
descriptor_set_layout_countThe number of descriptor set layouts.
descriptor_set_layoutsAn array of VkDescriptorSetLayout for the pipeline layout.
stage_countThe number of shader stages.
stagesAn array of VkPipelineShaderStageCreateInfo defining shader stages.
viewportThe VkViewport defining the viewport dimensions.
scissorThe VkRect2D defining the scissor rectangle.
is_wireframeTrue to use wireframe rasterization, False for fill mode.
out_pipelineA pointer to a vulkan_pipeline struct to populate with the created pipeline details.
Returns
True if the pipeline was created successfully, False otherwise.

Binds the specified pipeline to the given command buffer.

This function binds the provided vulkan_pipeline to the specified vulkan_command_buffer at the given pipeline bind point (graphics or compute). The command buffer must be in the recording state when this function is called.

Parameters
command_bufferA pointer to the vulkan_command_buffer to bind the pipeline to.
bind_pointThe VkPipelineBindPoint indicating whether it's a graphics or compute pipeline.
pipelineA pointer to the vulkan_pipeline to bind.

Destroys a Vulkan pipeline and frees its associated resources.

This function destroys the VkPipeline and VkPipelineLayout objects associated with the given vulkan_pipeline. It must be called before destroying the Vulkan device or context to avoid memory leaks.
