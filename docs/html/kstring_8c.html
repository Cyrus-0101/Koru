Detailed Description

Implementation of lightweight string utility functions.

Implements core string functionality used internally by the engine. Currently wraps standard C library functions but integrates with the engineâ€™s custom memory system.

Usage: These functions are primarily used by engine systems that need to:

  • Copy strings safely
  • Duplicate strings with engine-aware memory allocation
  • Interact with memory tracking via MEMORY_TAG_STRING
