Detailed Description

Utility functions for working with VkResult values.

This module provides:

  • Human-readable strings for VkResult codes
  • Helper function to check if a result is considered a success or failure

These are used throughout the engine to:

  • Log errors during initialization
  • Handle validation layer warnings
  • Debug GPU operations like image creation, command submission, etc.

b8 vulkan_result_is_success ( VkResult  result)

Checks whether a VkResult represents a successful operation.

Based on the Vulkan spec, some results are treated as "success" even if not VK_SUCCESS, such as VK_NOT_READY or VK_SUBOPTIMAL_KHR.

Parameters
resultThe VkResult to evaluate.
Returns
True if the result indicates success (operation completed), False otherwise (error occurred).
