Detailed Description

Image management utilities for the Koru Engine's Vulkan renderer.

This module provides functions to:

  • Create and allocate Vulkan images
  • Generate image views for use in render passes or shaders
  • Destroy images and associated resources

These are used throughout the rendering pipeline for color attachments, depth/stencil buffers, and texture resources.
