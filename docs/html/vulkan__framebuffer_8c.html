Detailed Description

Implementation of Vulkan framebuffer creation and destruction.

This module implements all functionality related to:

  • Allocating and copying image view attachments
  • Creating VkFramebuffer objects from render passes
  • Properly freeing memory during shutdown

Framebuffers are essential for rendering â€” they define what images are rendered into.

Creates a new Vulkan framebuffer using the provided attachments and render pass.

Wraps vkCreateFramebuffer(...) and stores the attachments internally. Used when creating or recreating framebuffers after swapchain changes.
