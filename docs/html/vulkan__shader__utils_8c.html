Detailed Description

Implements utility functions for loading and creating Vulkan shader modules.

This module provides functions to load SPIR-V shader binaries from files, create Vulkan shader modules, and set up pipeline shader stage info structures.

Creates a Vulkan shader module from a SPIR-V binary file.

This function loads the specified SPIR-V file, creates a VkShaderModule, and populates the corresponding vulkan_shader_stage structure with creation info and stage info for pipeline use.

Parameters
contextA pointer to the active Vulkan context.
nameThe base name of the shader (without extension).
type_strThe shader type string (e.g., "vert", "frag").
shader_stage_flagThe VkShaderStageFlagBits indicating the shader stage.
stage_indexThe index in the shader_stages array to populate.
shader_stagesAn array of vulkan_shader_stage to hold created stages.
Returns
True if the shader module was created successfully, False otherwise.
