Detailed Description

Lightweight string utility library.

This module provides basic string operations that are:

  • Safe(r) than raw C string functions
  • Integrated with the engine's memory system
  • Designed for clarity and maintainability

These utilities are intended to be used internally within the engine, not exposed to end-user game code (which should use standard C strings).
