Detailed Description

Vulkan command buffer management interface for the Koru Engine.

This module provides functions to:

  • Allocate and free command buffers from a pool
  • Begin and end command recording
  • Submit buffers to the GPU
  • Manage state transitions during the command buffer lifecycle

Command buffers store GPU commands like drawing, memory transfers, and render pass operations. They must be allocated from a command pool, recorded into, and submitted to a queue for execution.
