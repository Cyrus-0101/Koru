Detailed Description

Implementation of the tagged memory allocation system.

This module implements a memory tracking system that allows allocations to be categorized using tags (e.g., MEMORY_TAG_RENDERER, MEMORY_TAG_GAME). It provides:

  • Memory usage statistics per tag
  • Allocation and deallocation wrappers with tracking
  • Helper functions like kzero_memory(), kcopy_memory(), and kset_memory()
  • A utility function to generate formatted memory usage strings for debugging

The system wraps platform-specific memory functions (platform_allocate, platform_free) to ensure consistent behavior and logging.

Usage: Replace raw memory operations with kallocate() / kfree() to enable tracking.
