Detailed Description

Testbed application entry point implementation.

This file implements the required create_game() function used by the engine. It sets up the game configuration and allocates the game state.

Responsibilities:

  • Configure window settings
  • Assign game lifecycle function pointers
  • Allocate game-specific memory
