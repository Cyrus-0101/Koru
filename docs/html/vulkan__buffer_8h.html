Detailed Description

Vulkan buffer management interface for the Koru Engine.

This module provides functions to:

  • Create and destroy Vulkan buffers
  • Lock and unlock buffer memory for CPU access
  • Load data into buffers
  • Copy data between buffers

Buffers are used to store vertex data, index data, uniform data, and other GPU resources. Proper management of buffer memory is crucial for performance and correctness in Vulkan applications.
